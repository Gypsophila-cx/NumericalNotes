    \textbf{Note for FDM. Chapter 1:使用有限差分逼近导数}\\{\small Finite Difference Methods for Ordinary and Partial Differential Equations}

一般地，构造差分格式有以下几种方法：
\begin{enumerate}
    \item 待定系数法；
    \item 先使用多项式函数逼近原函数，再对多项式求导；
    \item 重复多次使用一阶差分格式。
\end{enumerate}

\hrule

\begin{center}\large
    \textbf{Example}
\end{center}

\begin{equation}
    D_+ f(x_i) = \frac{f(x_{i+1}) - f(x_i)}{x_{i+1} - x_i}
\end{equation}
